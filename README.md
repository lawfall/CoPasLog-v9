\# CoPasLog\_v9.7 - Smart Clipboard Manager



CoPasLog adalah aplikasi manajemen clipboard modern yang dirancang untuk meningkatkan produktivitas melalui fitur perekaman riwayat (history) teks secara otomatis dan pengisian formulir (autofill) yang cerdas. 



Aplikasi ini dikembangkan menggunakan Python dengan fokus pada kemudahan penggunaan, performa ringan, dan fungsionalitas \*Drag \& Drop\* yang stabil.



\## âœ¨ Fitur Utama

\- \*\*Auto-Recording\*\*: Mencatat setiap teks yang Anda salin (copy) ke dalam database lokal secara real-time.

\- \*\*Smart Drag \& Paste\*\*: Cukup klik dan tahan pada item riwayat, arahkan ke target, dan lepas untuk menempelkan (paste) secara otomatis.

\- \*\*Modern Dark UI\*\*: Tampilan minimalis bertema gelap yang nyaman di mata.

\- \*\*Search \& Manage\*\*: Pencarian cepat, penomoran urut, dan fitur hapus per item atau sekaligus.

\- \*\*Custom Transparency\*\*: Atur tingkat transparansi jendela agar tidak menghalangi pekerjaan Anda.

\- \*\*Export Data\*\*: Simpan riwayat clipboard Anda ke dalam format `.txt` melalui menu File.
